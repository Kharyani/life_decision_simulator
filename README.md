Life Decision Simulator (Desktop Application)
🚀 Overview

The Life Decision Simulator is a Python-based desktop application that helps users simulate how daily habits such as study hours and consistency impact long-term skill growth and career probability.

It uses a simple simulation model combined with visualization to show how different life choices can affect future outcomes.

🎯 Features
📥 User input for study hours and consistency
🧠 Life outcome simulation using probabilistic modeling
📈 Skill growth visualization using graphs
💼 Job probability estimation
🖥️ Simple and interactive desktop GUI
🛠️ Tech Stack
Python
PyQt5 (GUI Framework)
NumPy (Simulation logic)
Matplotlib (Data visualization)

📊 How It Works
User enters:
Study hours per day
Consistency level (0 to 1)
The system simulates skill growth over 180 days using random variation.
It calculates:
Skill progression
Job probability score
A graph displays the growth trend over time.
